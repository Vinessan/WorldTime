World Time

PT-BR
Criador: Vinicius Oliveira Santana
Esse site foi desenvolvido com o intuito e proposito de facilitar ao usuario uma maneira mais facil é rapida de ter acesso a Horas, Dia, Endereço IP e outras funcionalidades, de forma que essas informações venhao do mundo todo para que assim ele possa estar sempre atualizado, ou por curiosidade, obeter dados de outros continentes e paises que deseja. Como criador do site informo que: Por se tratar de uma API Global, em alguns momentos a chamada, a verificação da informação, podera demorar alguns segundos. Esse site foi um grande desafio e pude aprender muito com cada detalhe de seu código. Usei para desenvolve-ló, JavaScript para buscar informações na API (WorldTime), CSS para estilizar a pagina de acordo com o objetivo do site e HTML como base do desenvolvimento.

Creator: Vinicius Oliveira Santana
This site was developed with the intention and purpose of facilitating the user an easier way is quick to have access to Hours, Day, IP Address and other functionalities, so that this information comes from all over the world so that it can always be updated , or out of curiosity, obtain data from other continents and countries that you want.
